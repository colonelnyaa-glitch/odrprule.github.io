# ODRP Rules

GitHub Pages用の静的サイトです。カテゴリーは参考画像と同じ階層構成です。

- 基本ルール・禁止事項
- 犯罪関連
- 職業関連
- その他

`index.html`、`style.css`、`script.js` をリポジトリ直下へアップロードしてください。


## 車両価格一覧
`vehicle-prices.html` を車両価格カテゴリー内へ埋め込み済みです。検索・カテゴリ絞り込み・価格順・積載量順に対応しています。

## 管理画面
`admin.html` を開くと、サイトをプレビューしながら文章・リンク・画像パスを編集できます。
編集後は「index.htmlを書き出す」を押し、生成された `index.html` をGitHub上の元ファイルと置き換えてください。
